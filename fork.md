## Fork چیست؟

Fork یعنی گرفتن یک کپی از ریپازیتوری شخص یا سازمان دیگر در حساب GitHub خودمان.
وقتی یک پروژه را Fork می‌کنیم، نسخه‌ای مستقل از آن پروژه در پروفایل ما ساخته می‌شود.

### کاربرد Fork:
- مشارکت در پروژه‌های متن‌باز (Open Source)
- اعمال تغییر بدون دست زدن به پروژه اصلی
- تمرین و یادگیری روی کد دیگران

### مثال:
اگر بخواهیم در یک پروژه عمومی تغییر ایجاد کنیم، ابتدا آن را Fork می‌کنیم، تغییرات را انجام می‌دهیم و سپس Pull Request می‌فرستیم.
