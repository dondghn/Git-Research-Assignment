## Pull چیست؟

Pull به معنی دریافت آخرین تغییرات از ریپازیتوری راه‌دور (Remote) و اعمال آن‌ها روی نسخه محلی است.
این دستور معمولاً زمانی استفاده می‌شود که افراد دیگر روی پروژه تغییر ایجاد کرده‌اند.

### کاربرد Pull:
- به‌روز نگه داشتن پروژه محلی
- دریافت تغییرات جدید هم‌تیمی‌ها
- جلوگیری از تداخل نسخه‌ها

### مثال:
با اجرای دستور git pull، تغییرات جدید از GitHub دریافت و به پروژه اضافه می‌شود.
