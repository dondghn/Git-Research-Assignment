## Clone چیست؟

Clone یعنی کپی کردن یک ریپازیتوری از GitHub روی سیستم شخصی (کامپیوتر).
با Clone کردن، تمام فایل‌ها، تاریخچه تغییرات و شاخه‌ها (branches) دانلود می‌شوند.

### کاربرد Clone:
- کار کردن روی پروژه به صورت آفلاین
- ویرایش و توسعه پروژه
- اجرای پروژه روی سیستم شخصی

### مثال:
با دستور git clone می‌توان یک ریپازیتوری را از GitHub روی کامپیوتر کپی کرد.
