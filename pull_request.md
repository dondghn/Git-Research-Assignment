## تفاوت Pull، Pull Request و Push

### Pull:
Pull برای دریافت تغییرات از ریپازیتوری راه‌دور به سیستم محلی استفاده می‌شود.
این کار معمولاً توسط توسعه‌دهنده انجام می‌شود.

### Push:
Push برای ارسال تغییرات از سیستم محلی به ریپازیتوری GitHub استفاده می‌شود.

### Pull Request:
Pull Request برای درخواست رسمی جهت اضافه شدن تغییرات به پروژه اصلی است و نیاز به تأیید دارد.

### خلاصه تفاوت‌ها:
- Pull: دریافت تغییرات
- Push: ارسال تغییرات
- Pull Request: درخواست بررسی و ادغام تغییرات
